question: "Qu'est-ce qui se passe si Cloudflare ou GitHub change ses prix ?"
answer: "C'est ta responsabilité après la livraison, puisque les comptes sont à ton nom. Mais pour te rassurer : à ce jour, GitHub et Cloudflare sont gratuits pour l'usage d'un site vitrine pro, et le resteront très probablement (ce sont leurs offres d'appel). Si un jour ça change, tu auras le code source et tu pourras migrer facilement chez un autre hébergeur gratuit."
